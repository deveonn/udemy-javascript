Teste de Prompt, Alert, e Confirm

Alert

Apenas uma caixa de alerta para gerar algum aviso, não retorna nada.

Confirm

Diferente de alert, pode retornar True ou false.

Prompt

Consegue retornar o que o usuário escreve no campo input que aparece.
